# Forecast

Мой учебный проект. APK доступно в релизе.

Использованные технологии: <br />
• Kotlin <br />
• MVVM <br />
• Retrofit <br />
• Dagger2 <br />
• Room <br />
• Coroutines <br />
• Flow <br />
• LiveData <br />
• ViewPager2 <br />
• RecyclerView + CardView <br />
• Немного рисования на канвасе <br />
• Shimmer effect <br />
• Паттерн Single Activity <br />
